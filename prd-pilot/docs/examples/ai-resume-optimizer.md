# AI 简历优化工具

## 1. 输入

- 产品名称：AI 简历优化工具
- 产品类型：Web 工具 / AI 辅助写作
- 核心目标：帮助用户上传或输入简历内容，快速生成优化建议、改写版本和岗位匹配建议。
- 重点诉求：内容简洁、优化建议可解释、结果可直接复制使用。

## 2. Requirement Spec

- 目标用户：求职学生、转行者、应届毕业生、自由职业者
- 用户痛点：
  - 简历内容平淡，缺少亮点
  - 不知道如何针对岗位调整表达
  - 需要快速得到可复制的修改版本
- 核心场景：
  - 粘贴简历文本后生成优化建议
  - 针对某个岗位调整简历
  - 查看关键词覆盖和表达强度
  - 将优化后的版本导出
- 关键功能：
  - 简历导入与编辑
  - 问题诊断与优化建议
  - 岗位匹配提示
  - 一键生成改写版本
- 主要页面：
  - 简历输入页
  - 优化建议页
  - 岗位匹配页
  - 导出结果页
- 原型风格：移动端 H5 + 简洁工具感
- 成功标准：
  - 用户能在一次会话内完成“输入 -> 诊断 -> 优化 -> 导出”
  - Demo 中能演示针对岗位的定点调整

## 3. PRD 要点

- 输入区支持文本粘贴、分段输入和岗位描述
- 优化区把问题分成表达、结构、关键词、成果量化四类
- 岗位匹配区展示匹配分数和缺失关键词
- 导出区支持复制、下载和版本对比

## 5. 一次修改结果

反馈示例：

> 诊断结果太抽象了，希望把“问题”直接映射到“可修改的句子”，并在岗位匹配页突出缺失关键词。

修改结果：

- 在诊断结果中增加“可改写句子”区域
- 把建议分成“结构、表达、量化、关键词”四类
- 岗位匹配页增加缺失关键词列表和优先级提示
- 导出页保留优化前后对比，方便用户直接复用
