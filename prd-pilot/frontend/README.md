# PRD Pilot Frontend

Vue 3 + Vite single-page application for the PRD Pilot workspace.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Local URLs

- Frontend: `http://localhost:5173`
- Backend API: `http://localhost:8000`

## Notes

- Model configuration can be edited directly in the UI.
- API keys entered in the page are stored only in the current browser.
- Build output is generated into `dist/` and should not be committed.
